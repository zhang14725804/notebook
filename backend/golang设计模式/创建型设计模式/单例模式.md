## 单例模式

### 1.懒汉模式。

```golang
type Singleton struct{}
// private
var instance *Singleton
// public
func GetInstance() *Singleton{
    // 😅 非线程安全
    if instance == nil{
        instance = &Singleton{}
    }
    return instance
}
```

###  2. 懒汉加锁模式

```golang
type Singleton struct{}
// private
var instance *Singleton

var mu sync.Mutex
// public
func GetInstance() *Singleton{
    // 😅 这种结构的单例模式,会导致每次都加锁,严重浪费性能
    mu.Lock()
    defer mu.Unlock()
    if instance == nil{
        instance = &Singleton{}
    }
    return instance
}
```

###  3. 懒汉双重锁检测模式

```golang
type Singleton struct{}
// private
var instance *Singleton

var mu sync.Mutex
// public
func GetInstance() *Singleton{
    // 😅 先判断
    if instance == nil{
        // 再加锁
        mu.Lock()
        defer mu.Unlock()
        // 再判断
        if instance == nil{
            instance = &Singleton{}
        }
    }
    return instance
}
```

### 4. 利用内置的sync.Once来实现

```golang
type Singleton struct{}
// private
var instance *Singleton

var lock sync.Once

func GetInstance() *Singleton{
    lock.Do(func(){
        instance = &Singleton{}
    })
    return instance
}
```
